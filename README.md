## Object Literal

 * If Key-Value pair where `Key` & `Value` is identical, we can condense as one.

   *Before*
    ````
    inventory: inventory
    ````

    *After*
    ````
    inventory
    ````
 * If Key-Value pair where `Value` is `function`, we can omit the `function keyword and  colon (:)`

    *Before*
    ````
    priceForTitle: function(title){
    }
    ````
    *After*
    ````
    priceForTitle(title){
    }
    ````
